# 第二节 文件和目录相关命令：less

## 1、简介

命令：less

作用：分屏查看文件内容



## 2、用法

### ①使用less命令打开文件

> [root@localhost ~]# less long



### ②less环境

![images](images/img035.png)



### ③控制方式

| 按键    | 效果                           |
| ------- | ------------------------------ |
| 空格    | 向下滚动一屏                   |
| b       | 向上滚动一屏                   |
| 回车    | 向下滚动一行                   |
| q       | 退出                           |
| /关键词 | 搜索<br />n向下找<br />N向上找 |
| =       | 显示详细的文件信息             |
